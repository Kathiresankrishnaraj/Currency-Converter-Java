
# Currency Converter (Java)

A simple console-based Java application to convert Indian Rupees (INR) into
USD, EUR, and GBP using menu-driven programming.

## Features
- INR to USD, EUR, GBP conversion
- Menu-driven console application
- Uses switch-case and methods

## Technologies Used
- Java
- Scanner
- OOP concepts

## How to Run
1. Compile the program:
   javac CurrencyConverter.java
2. Run the program:
   java CurrencyConverter
